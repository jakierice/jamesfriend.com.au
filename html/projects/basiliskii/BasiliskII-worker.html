Basilisk II Classic Mac emulator

This is the Basilisk II Classic Mac emulator running Mac OS System 7, with SimCity 2000 and Bungie Software's Marathon. A hack by @ur_friend_james

This emulator should work in Chrome 69 and 72 (but not Chrome 70 due to a bug in that version). It should work in Firefox if you enable the javascript.options.shared_memory flag in the about:config page.

This application requires a browser which supports SharedArrayBuffer (the emulator runs in a Web Worker and communicates with the main thread via shared memory).
